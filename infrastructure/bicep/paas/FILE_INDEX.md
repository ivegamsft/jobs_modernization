### Configuration Files

#### ‚úÖ `parameters.bicepparam` (3.3 KB, 63 lines)

**Purpose:** Master parameter template with sample values

**What it contains:**

- Environment settings (environment, applicationName, location)
- App Service configuration (appServiceSku)
- SQL Database configuration (edition, objective, admin credentials)
- Core infrastructure parameters (all 7 parameters with documentation)
- Placeholder values for core module outputs
- Clear comments for each parameter

**Use when:**

1. Creating new deployment parameter files
2. Understanding required parameters
3. Documenting parameter values for teams

**Before deployment:**

- Copy this file or use as reference
- Update all parameter values
- **CRITICAL:** Change sqlAdminPassword from default
- **CRITICAL:** Paste actual core infrastructure IDs

**Security notes:**

- Don't commit with real credentials
- Use Azure Key Vault for sensitive values in production
- Mark as `@secure()` parameters in template
