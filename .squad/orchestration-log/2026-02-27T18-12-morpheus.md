## Blockers & Mitigations

**Critical:**
- App won't build → Detailed NuGet + framework verification
- Database won't deploy → SqlPackage.exe fallback + DACPAC steps
- Connection string mismatch → Explicit update instructions

**Medium:**
- Port 8080 in use → Use alternate port
- Session database issues → Verify DACPAC includes membership
- File locking → Use SSMS to detach before rebuild
